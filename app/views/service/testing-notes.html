Testing notes

To help find accessibility issues you could use the WAVE browser extension for Chrome or Firefox.

Please note that WAVE, as with other automated evaluation tools, will not tell you all the accessibility issues that occur on a web page. They can only test some of the accessibility requirements that do not need human intervention. Testing done by humans is essential to ensure an accessible user experience.

Testing checklist

  • Does WAVE find any errors?
  • Do page titles make sense?
  • Are headings used correctly?
  • Can you access everything using the keyboard?
  • Are form fields labeled correctly?
  • Are there any colour contrast errors or reliance on colour alone?

The NHS Digital service manual has further guidance on accessibility testing which you could refer to for more checks.
